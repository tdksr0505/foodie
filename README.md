# 餐廳收藏系統

## demo:

[MoguMogu](https://mogumogu.vercel.app/)

## 功能

- 收藏紀錄 想吃/已吃過 的餐廳
- 可以從離餐廳最近捷運站及餐廳類型去做篩選查詢
